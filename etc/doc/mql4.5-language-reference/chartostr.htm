CharToStr

Conversion of the symbol code into a one-character string.

string  CharToStr(
   uchar  char_code     // ASCII-code
   );

Parameters

char_code

[in]  ASCII char code.

Returned value

Text string.

Example:

  string str="WORL" + CharToStr(68); // 68 is code for 'D'
  // the resulting string will be WORLD
